# Markdown Viewer/Editor

A simple and lightweight markdown viewer and editor for real-time preview and editing.

## Features

- **Live Preview**: View markdown as you type
- **Syntax Highlighting**: Code blocks with language support
- **Export Options**: Save as HTML or PDF
- **Responsive Design**: Works on desktop and tablet devices
- **Keyboard Shortcuts**: Quick formatting with hotkeys

## Installation

```bash
git clone https://github.com/yourusername/markdown-editor.git
cd markdown-editor
npm install
```

## Usage

Navigate to: https://danbagan-markdown-viewer.netlify.app

[![Netlify Status](https://api.netlify.com/api/v1/badges/3a9e3567-e2f8-41e7-91c1-58dc9263e68b/deploy-status)](https://app.netlify.com/projects/danbagan-markdown-viewer/deploys)

Open your browser and navigate to `http://localhost:3000`

## Keyboard Shortcuts

| Shortcut | Action  |
| -------- | ------- |
| `Ctrl+B` | Bold    |
| `Ctrl+I` | Italic  |
| `Ctrl+H` | Heading |
| `Ctrl+S` | Save    |
